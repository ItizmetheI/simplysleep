# Simply Rest Phase 1 Redirect Rules

## TL;DR

The canonical product-review base for this packet is `/mattress-reviews/{product}/`. Do not allow `/reviews/{product}/` and `/mattress-reviews/{product}/` to both index.

Live check on 2026-06-25 found `/mattress-reviews/amerisleep-as3/` redirecting to `/best-online-mattress/`. Remove that redirect before publishing the AS3 review.

## Required Fixes

1. Remove or override the current redirect:

```text
/mattress-reviews/amerisleep-as3/ -> /best-online-mattress/
```

2. Keep this as the canonical AS3 review URL:

```text
https://simplyrest.com/mattress-reviews/amerisleep-as3/
```

3. If `/reviews/amerisleep-as3/` exists or is generated by WordPress, redirect it here:

```text
/reviews/amerisleep-as3/ -> /mattress-reviews/amerisleep-as3/
```

4. For future product reviews, redirect the old base to the canonical base:

```text
/reviews/{slug}/ -> /mattress-reviews/{slug}/
```

## WordPress Redirection Plugin

Use these as Redirection plugin rules:

| Source URL | Target URL | Regex | HTTP Code |
|---|---|---:|---:|
| `^/reviews/(.+)/?$` | `/mattress-reviews/$1/` | Yes | 301 |
| `/mattress-reviews/amerisleep-as3/` | Remove any target to `/best-online-mattress/` | No | N/A |

## WP-CLI Cleanup Helper

The deploy bundle includes a dry-run-first cleanup helper for common WordPress redirect plugins:

```bash
wp eval-file simplyrest_phase1_as3_redirect_cleanup.php
wp eval-file simplyrest_phase1_as3_redirect_cleanup.php -- --apply
```

Only run `--apply` after database backup and approval. The helper disables records only when the row clearly contains both `amerisleep-as3` and `best-online-mattress`. It does not touch server, CDN, or htaccess redirects.

## Apache / htaccess

Add only after confirming WordPress is not already managing the same redirect:

```apache
RedirectMatch 301 ^/reviews/(.+)/?$ /mattress-reviews/$1/
```

Do not add a redirect from `/mattress-reviews/amerisleep-as3/` to any other page.

## Nginx

```nginx
rewrite ^/reviews/(.+)/?$ /mattress-reviews/$1/ permanent;
```

Do not add a redirect from `/mattress-reviews/amerisleep-as3/` to any other page.
